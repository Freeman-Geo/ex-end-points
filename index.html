Crear usuarios en thunderCLient


Tecnicamente es en el body en formato JSON, usando el verbo POST, por ejemplo...


http://localhost:3000/account

{
"guid": "123",
"name": "Geo Freeman"
}


Una vez hecho eso, nos volvemos a pasar al verbo GET, y en el link despues de account, le metemos el id del new commer.


http://localhost:3000/account/123

{
"id": "123",
"name": "Geo Freeman"
}


así nos regresa el mismo json pero como respuesta.
- Ahora, usando el verbo Patch, en el body le voy a cambiar de nombre a Geo
Usaré el verbo Patch, y elmismo link -> http://localhost:3000/account/112233


{
"name": "Geo Freeman"
}


Usuario actualizado


Y ahora le vuelvo a dar al verbo GET


{
"name": "Freeman"
}


Usuario actualizado


Y ahora le doy en el verbo GET puedes dejar o quitar el mini-código del body y le doy send


{
"id": "112233",
"name": "Freeman"
}


Y asi se pueden manejar los verbos en las solicitudes HTML
